# Stellaris-Galaxy-position-swap-
调换《群星》存档中两个星系的位置Galaxy position swap
使用方法，首先找到游戏存档，不出意外你可以在\Documents\Paradox Interactive\Stellaris\save games，或者 文档\Paradox Interactive\Stellaris\save games找到它。
找到你本地的存档文件，大概是以.sav结尾的文件。使用任意解压工具解压。
你会得到一个gamestate文件。
现在启动这个python脚本，输入你想要交换的两个恒星系的id就可以交换了，id怎么看你可以游戏控制台用debugtooltip查看。
我建议第一个id输入较小的那个，第二个id输入较大的那个。
最后把gamestate和meta压缩成zip，把后缀名改成.sav就可以了。
To use the Galaxy position swap method in the Stellaris save file, first locate the game save file. If everything goes as expected, you can find it in \Documents\Paradox Interactive\Stellaris\save games, or Documents\Paradox Interactive\Stellaris\save games.
Locate your local save file, which is likely to end with a .sav extension. Use any unzipping tool to decompress it.
You will obtain a gamestate file.
Now, start this Python script and input the IDs of the two star systems you want to exchange. You can view the IDs by using the debug tooltip in the game console.
I suggest entering the smaller one for the first ID and the larger one for the second ID.
Finally, compress gamestate and meta into a zip file, and change the extension to .sav.
